# homebrew-livegit

Homebrew tap for [`lg` (Live Git)](https://github.com/iamtaehyunpark/livegit) —
work on your remote GPU/lab server as if its files were on your laptop.

## Install

```sh
brew tap iamtaehyunpark/livegit
brew install lg
```

The formula downloads a prebuilt static binary from the
[`livegit` releases](https://github.com/iamtaehyunpark/livegit/releases) — you
don't need Go to install or run `lg`.
